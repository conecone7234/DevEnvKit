# VScodeEnvKit

[English README](https://github.com/conecone7234/VScodeEnvKit/blob/main/README.md)

複数言語のポータブル開発環境をワンタッチで構築するスクリプト集です。  
クローンして各ブランチの `setup.bat` を実行するだけで、必要な言語ランタイムと VSCode 設定が整います。
